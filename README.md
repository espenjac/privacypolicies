# Privacy Policies

A repository for hosting privacy policies via GitHub Pages.

## Overview

This repository is configured to host privacy policies using GitHub Pages. The site is automatically published when changes are pushed to the main branch.

## How to Use

1. Add your privacy policy HTML or Markdown files to this repository
2. Update the `index.html` to link to your privacy policies
3. Push changes to the main branch
4. Your privacy policies will be available at `https://<username>.github.io/privacypolicies/`

## Files

- `index.html` - Main landing page listing all available privacy policies
- `_config.yml` - Jekyll configuration for GitHub Pages

## Adding a New Privacy Policy

1. Create a new HTML or Markdown file (e.g., `myapp-privacy-policy.html`)
2. Add a link to the new policy in `index.html`
3. Commit and push your changes

## GitHub Pages Setup

To enable GitHub Pages for this repository:
1. Go to repository Settings
2. Navigate to Pages section
3. Under "Source", select the branch to deploy from (usually `main`)
4. Save the settings

Your site will be published at `https://<username>.github.io/privacypolicies/`
